# 🩷 프로젝트 개요

### 🌸 PiBoo: 성분부터 궁합까지, 똑똑한 화장품 도우미
[고치기] 이 챗봇은 사용자의 피부 타입, 고민, 보유 중인 제품 등을 바탕으로 궁합이 잘 맞는 기초 화장품이나 스킨케어 제품을 추천해줍니다. 제품 성분과 실제 사용자 리뷰까지 반영해, 더 신뢰할 수 있는 맞춤형 정보를 제공합니다.
